## CRM Site Listings (Notion)

Commercial property database for the Shanlan Tea House search.

**Database**: `site_listings`

**Fields**: Site ID | Site Name | Area | Rent | Location | Floor | MEP | Accessibility | Competitors | Notes | Status

## Footfall and Rent Tracker (Google Sheet)

Monthly footfall and rent data for all listed sites.

**Sheet**: `Footfall_and_Rent`

**Columns**: Site ID | Monthly Footfall | Rent per Month | Efficiency

## File System

- `input/brand_brief.pdf` — Brand requirements document (read-only)
- `input/founder_voice.mp3` — Founder's voice memo (read-only)
- `input/founder_voice_transcript.txt` — Voice memo transcript (read-only)
- `input/site_materials/` — Site photos, floor plans, MEP documents (read-only)
- `outputs/` — Agent output area (read-write). Place all deliverables here.
