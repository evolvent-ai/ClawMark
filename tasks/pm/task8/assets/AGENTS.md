# Work Standards

## Weekly Report Output
- Fill in the `weekly_report_template.csv` template in the workspace, output to `output/weekly_report.csv`
- See the template file header comments for filling rules

## Tool Usage
- **Feishu**: View team group chat messages (text, screenshots, voice)
- **Email**: Receive and send formal communications (reports, attachments, etc.)
- **Notion**: Project requirement pool `req_pool_v3`
- **Google Sheets**: Team schedule `cloudsail_schedule_v3`
- **Filesystem**: Reference files and output directory in the workspace

## Information Sources
- Work involves multiple information sources; information in these systems may change at any time
- Different sources may have time lags or inconsistencies; you need to make your own judgments
