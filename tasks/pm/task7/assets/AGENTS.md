# Work Guidelines

## Procurement Evaluation Output
- Fill in according to the `procurement_template.xlsx` template in workspace, output to `output/procurement_evaluation.xlsx`
- See Sheet3 "Instructions" in the template for filling rules

## Tool Usage
- **Email**: Receive formal quotation emails (with PDF attachments), send approval emails to the director
- **Notion**: Procurement Tracking Dashboard `yunfan_procurement_q2`
- **Google Sheets**: Vendor Historical Evaluation data `vendor_history_2025`
- **Google Calendar**: Create procurement review meetings
- **Filesystem**: Reference files and output directory in workspace

## Information Sources
- Your work involves multiple information sources, and the information in these systems may change at any time
- Different sources may describe the same vendor with time lags or differing perspectives; you need to exercise your own judgment
